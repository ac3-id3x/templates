Comment ajouter un visuel ?

Pour ajouter un visuel vous devez cliquer sur "Parcourir..." afin d'indiquer où se trouve le visuel sur votre ordinateur.
Le chemin du fichier doit s'afficher à gauche du bouton "Parcourir...".
Vous devez ensuite cliquer sur "Envoyer le visuel".
Durant quelques secondes, votre visuel sera sous la forme d'une image avec le message "PHOTO DISPONIBLE PROCHAINEMENT".

Caractéristiques

 • Format : jpg
 • Autres formats acceptés : jpeg, gif, bmp, tif, tiff et png
 • Tailles min : 100 pixels * 100 pixels
 • Tailles max : 800 pixels * 600 pixels
 • Poid max : 512Ko
