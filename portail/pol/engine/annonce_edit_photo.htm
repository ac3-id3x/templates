Aide
Comment ajouter une photo ?

Pour ajouter une photo vous devez cliquer sur "Parcourir..." afin d'indiquer où se trouve la photo sur votre ordinateur.
Le chemin du fichier doit s'afficher à gauche du bouton "Parcourir...".
Vous devez ensuite cliquer sur "Envoyer la photo".
Durant quelques secondes, votre photo sera sous la forme d'une image avec le message "PHOTO DISPONIBLE PROCHAINEMENT".

Caractéristiques

  • Format : jpg
  • Autres formats acceptés : jpeg, gif, bmp, tif, tiff et png
  • Tailles min : 100 pixels * 100 pixels
  • Tailles max : 800 pixels * 600 pixels
  • Poid max : 512Ko
