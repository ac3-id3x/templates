Comment ajouter une photo 360° ?

Pour ajouter une photo 360° vous devez cliquer sur "Parcourir..." afin d'indiquer où se trouve la photo sur votre ordinateur, le format accepté est .jpg, le rapport hauteur/largeur doit être de 16/9.
Le chemin du fichier doit s'afficher à gauche du bouton "Parcourir..."
Vous devez préciser si l'image est de type "cylindre" ou "sphère"
Vous devez ensuite cliquer sur "Valider".
