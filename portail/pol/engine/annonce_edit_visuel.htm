Comment ajouter une photo ?
Pour ajouter une photo vous devez cliquer sur "Parcourir" afin d'indiquer où se trouve la photo sur votre ordinateur, le format accepté est .jpg, la taille maximale est de 800px sur 600px, le poid maximal de 512Ko.
Le chemin du fichier doit s'afficher à gauche du bouton "Parcourir"
Vous devez ensuite cliquer sur "Envoyer la photo".
La page principale de l'annonce affichera alors, durant quelques secondes, votre photo sous la forme d'une image "PHOTO DISPONIBLE PROCHAINEMENT"
