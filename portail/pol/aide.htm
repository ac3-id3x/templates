Pour un fonctionnement optimal, ce site nécessite la mise à jour :

• De votre navigateur
Téléchargez Internet Explorer 7 
Téléchargez la dernière version Firefox (pour Windows et Mac)

• Du lecteur Flash 8 ou supérieur
Télécharger Flash Macromedia

• Pour les utilisateurs de Microsoft Windows, il est conseillé d'activer Windows Update afin de profiter au mieux des produits Microsoft. Les mises à jours de Microsoft Windows sont aussi disponibles via update.microsoft.com.


La configuration suivante doit être mise en place :

• Votre navigateur doit accepter les cookies.
• Le javascript de votre navigateur doit être activé.
• Le cache de votre navigateur doit être désactivé. $$INCL:BASPOP.HTM$$
