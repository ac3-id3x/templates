Procédure

Conditions de reprise de vos documents personnalisés

  • Envoi des fichiers par email en une seule fois sous forme de fichier compressé (.zip) à l'adresse evoluervers5@poliris.fr
  • Le nom du fichier compressé doit comporter le code STE SITE de l'agence (858501.zip ou 858501laforetville.zip)
  • Date limite de réception des fichiers :  le 25 juillet 2014

La restitution sera intégrée dans votre bibliothèque de documents le jour de l'activation de Périclès5.

Le support technique Périclès est à votre entière disposition au 01 56 41 09 00 pour tous compléments d’information et vous assister si nécessaire pour extraire ces documents

 

Prérequis matériel

Configurations requises

  • Un ordinateur équipé d'un processeur Intel Core 2 Duo, Core i3, Core i5, Core i7, ou Xeon (les processeurs Core Solo et Core Duo ne sont plus prise en charge pour les Macs)
  • Un pavé Numérique. (Pour l’Avis de Valeur)
  • 2 Go de mémoire minimum, 4 Go de mémoire ou plus conseillé.
  • Windows Vista, Windows 7 Windows 8. Les versions Pro sont conseillées.
  • Pour les Macs voir préconisation Périclès Net sous MAC
  • Résolution minimale : 1280 X 1024 pour un écran 4/3 et de 1280 X 900 pour les écrans 16/9

Ce qui doit-être installé sur l’ordinateur

  • Windows à jour
  • Net Framework 4
  • Silverlight 5
  • Adobe Reader
  • Pdf Créator
  • Internet Explorer 8 ou plus

Vérification des pré-requis

A partir de votre écran Windows Cliquer droit sur "Poste de travail" ou récemment "Ordinateur" puis "propriétés". Vous obtiendrez une fenêtre telle que ci-dessous regroupant les informations.
