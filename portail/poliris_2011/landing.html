Sur la mise à jour d’Avis de Valeur

Avis de Valeur est la solution d’estimation de référence pour les professionnels de l’immobilier. Comme tout outil technologique, Avis de Valeur évolue pour devenir encore plus incontournable dans la prise de mandat et vous faciliter encore plus votre utilisation.

La mise à jour d’Avis de Valeur apporte plusieurs innovations qui le rendent encore plus utile au quotidien. Toutes les fonctionnalités qui ont fait le succès d’Avis de Valeur sont maintenues. La mise à jour les renforce. On peut noter :
1. Une refonte de l’interface pour une plus grande simplification d’utilisation
2. Une compatibilité avec les appareils mobiles grâce au responsive design
3. La refonte du module d’édition des rapports pour plus de personnalisation
4. Un meilleur outil de cartographie pour une analyse comparative plus juste

Pour les clients Périclès (Périclès 5 ou Périclès Air), la mise à jour aura lieu le 28 octobre 2015. Elle sera automatique pour tous les utilisateurs. Pour les clients SeLoger, la mise en place se fera fin novembre. La date exacte vous sera prochainement communiquée.
