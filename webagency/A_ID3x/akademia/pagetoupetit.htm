Ce cycle d'une année permet au tout-petit de se familiariser avec un répertoire iconographique qui l'aidera à poursuivre ses aquisitions dans l'Histoire des grandes civilisations.

Construit avec un thème par trimestre, les séances se déroulent comme suit :

  • 1er trimestre :
    L'animal dans l'art
  • 2ème trimestre :
    L'homme dans l'art
  • 3ème trimestre :
    Le paysage dans l'art
