  • Contrôle amiante obligatoire si bien construit avant 1997

LE CONTRÔLE DE L’ASSAINISSEMENT :

  • Dans les cas d’un assainissement individuel, un contrôle de conformité de moins de 3 ans est obligatoire
  • Diagnostic Performance Energétique obligatoire dans tous les cas
  • Si l’assainissement est à refaire, une étude de sol est également obligatoire pour la mise aux normes que doit légalement faire l’acquéreur dans un délai d’un an suivant l’achat
  • Dans le cas d’un assainissement collectif, un contrôle de conformité du raccordement est désormais demandé par les Notaires à chaque signature

Pour chacune de ces démarches, notre agence se déplace, prend les RDV, vous accompagne et vous conseille afin que votre dossier soit complet pour la mise en vente.
