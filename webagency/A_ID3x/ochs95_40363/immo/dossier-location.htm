Dossier location

Vous cherchez un logement

Pour toute demande de location, nous vous remercions de nous faire parvenir les pièces justificatives
ci-dessous :

Pour tous :

  • • Fiche de renseignements complétée et signée
  • • copie recto verso de la carte nationale d’identité ou du passeport
  • ou de la carte de séjour (en cours de validité)
  • • Simulation d’APL sur le site de la CAF (si bénéficiaire)
  • • les trois dernières quittances de loyer ou taxe foncière ou attestation d’hébergement
  • • un R.I.B (si prélèvement)

Pour les salariés :

  • • Les trois derniers bulletins de salaire
  • • Une attestation d’emploi datant de moins d’1 mois mentionnant le type de contrat,
  • la rémunération et la présence dans l’entreprise.
  • • Copie du contrat de travail signé
  • • Le dernier avis d’impôts sur les revenus (toutes les pages)
  • • Copie de la dernière déclaration de revenu (en période de déclaration)

Pour les retraités :

  • • Les 2 derniers avis d’impôts sur les revenus
  • • Notifications de retraite et complémentaires si retraité depuis moins de 2 ans

Pour les indépendants (statut gérant, libéral…) :

  • • Extrait de K-bis ou carte professionnelle ou certificat d’inscription au registre du commerce
  • • Les 2 derniers avis d’impôts sur les revenus
  • • les 2 derniers bilans (minimum 2 ans d’activité)

Merci de nous transmettre les pièces par mail : ochs@ochs-gerance.fr
