///////////////////

INTERVENIR DAS LA FEUILLE DE STYLE

* Body : la police ne change jamais

* #Flag :Pour paramétrer la distance entre le côté gauche  du site et les drapeaux, si langues

* #Header : Il existe deux id Header, un pour le flash et un no flash activé par défaut. Il suffit de modifier les commentaires pour la version avec flash.

* #Block_menu a : Pour modifier la hauteur des onglets de la navigation verticale, il suffit de modifier le padding dans cette id. Utile dès que le nombres d'onglets est différent de celui de base.

* # Menu_bottom : C'est le menu de navigation horizontale. Mettre 7690b5 en arrière-plan  si Flash et background-transparent si noflash

* #Menu_bottom a,.SepSit : En modifiant le margin-left, on modifie la distance entre les onglets de la navigation horizontale

* #TexteIntro :  C'est dans la div correspondant à cette id qu'on place le texte d'intro 


//////////////////

* La div flash est en commentaire html. Placer la avec les propriétés right et top dans la ccs (#Flash)

* Ajouter dans la div Flash pour le réf :$$INCL:TEXTE_VILLES.HTM$$
